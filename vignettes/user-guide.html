Color guide

ggplot(sun.spct) +
  geom_line() + 
  stat_color_guide(w.band = VIS_bands(), ymin = -0.03, ymax = 0) + 
  scale_fill_identity() +
  theme_bw()

which can be simplified by use of color_guide() which adds both the statistics and the scale. We addionaly leave ymin at its default of -Inf so that it is clipped at the edge of the plotting area.

ggplot(sun.spct) +
  geom_line() + 
  color_guide(w.band = VIS_bands(), ymax = -0.025) + 
  theme_bw()

ggplot(sun.spct) +
  geom_line() +
  color_guide(ymax = -0.025) + 
  theme_bw()

ggplot(sun.spct) +
  color_guide(alpha = 0.4) + 
  geom_line() +
  theme_bw()
